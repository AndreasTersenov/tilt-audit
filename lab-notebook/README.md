# The lab notebook

This directory is the as-it-happened record of the four run nights — kept
verbatim, including the mistakes, because the process is part of the result.
Every experiment in this repo was **pre-registered**: predictions frozen in
[`../RESEARCH_LOG.md`](../RESEARCH_LOG.md) with confidence levels and
resolution criteria, pushed publicly *before* the first GPU job of each night,
then scored afterwards — hits, misses, and self-corrections alike.

Chronology:

| Night | Log (as-it-happened) | Morning handoff (scored) |
|---|---|---|
| 2026-07-02→03 · GRF pilot | `NIGHT_LOG.md` | `HANDOFF_DAWN.md` |
| 2026-07-03→04 · arms night | `NIGHT_LOG_2026-07-04.md` | `HANDOFF_DAWN_2.md` |
| 2026-07-04→05 · certifier trial + transfer | `NIGHT_LOG_2026-07-05.md` | `HANDOFF_DAWN_3.md` |
| 2026-07-05 · day session (K-vs-2K, NFE, re-verification) | in `../RESEARCH_LOG.md` | — |

Tags used in the night logs: `[JOB]` launches · `[GATE]` correctness gates ·
`[RESULT]` findings · `[STEER]` live decisions · `[FAIL]` mistakes and
incidents (kept, not edited) · `[NOTE]` context.

The frozen overnight plans the logs execute are in [`../docs/`](../docs/).
